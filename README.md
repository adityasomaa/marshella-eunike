# Marshella Eunike — Beauty Brand Proposal · 5 Konsep Preview

Proposal kerja desain web untuk beauty brand client. 5 konsep arah desain yang berbeda — pilih satu, atau remix dari beberapa.

## 5 Konsep

| # | Brand | Mood | Folder |
|---|---|---|---|
| 01 | **AURA NOIR** | Dark luxury — Tom Ford / Byredo | `/brand-01-aura-noir` |
| 02 | **DEWY** | K-beauty glass skin — Tatcha / Glossier | `/brand-02-dewy` |
| 03 | **TERRA BOTANICA** | Botanical apothecary — Aesop | `/brand-03-terra` |
| 04 | **PRISM** | Holographic Gen-Z futurism | `/brand-04-prism` |
| 05 | **ROUGE** | Editorial Parisian — Vogue mood | `/brand-05-rouge` |

Setiap brand punya 7 halaman fungsional: `index.html`, `shop.html`, `product.html`, `about.html`, `contact.html`, `cart.html`, `checkout.html`.

## Tech stack (demo)

- **Bootstrap 5.3** — grid + utility classes via CDN
- **GSAP 3.12 + ScrollTrigger** — reveals, magnetic CTAs, scroll-triggered animations
- **Canvas API** — motion background unik per brand (mouse-reactive, 60fps)
- **Inline SVG + CSS** — semua placeholder graphic (zero external images)
- **Google Fonts** — Cormorant, Fraunces, DM Serif, Space Grotesk, Bodoni Moda

## Production-phase

- **CMS / Commerce**: Drupal 10 + Drupal Commerce
- **Frontend**: Bootstrap 5 (twig template integration)
- **Deploy**: Vercel edge + GitHub CI/CD
